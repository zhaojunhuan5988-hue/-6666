# 工厂车间ERP系统 — 使用说明

## 快速启动

### 方法一：直接运行（推荐）
```bash
cd factory-erp
python app.py
```
浏览器访问：http://localhost:5000

### 方法二：手机/其他电脑访问
1. 先用 `ipconfig`（Windows）或 `ifconfig`（Linux/Mac）查看电脑的局域网 IP
2. 其他设备浏览器输入：http://电脑IP:5000

### 默认账号
- 用户名：`admin`
- 密码：`admin123`
- 登录后请在"系统用户"中修改默认密码

## 功能模块

### 1. 数据看板（首页）
- 今日生产概况（待生产/生产中/已完成）
- 设备运行状态统计
- 库存预警列表
- 快速操作入口

### 2. 生产管理
- 创建生产工单（产品名称、计划数量、负责人等）
- 工单状态流转：待生产 → 生产中 → 已完成
- 按状态筛选工单
- 编辑和删除工单

### 3. 库存物料
- 物料台账管理（新增、编辑、删除物料）
- 入库/出库操作
- 出入库记录查询
- 库存预警（低于最低库存自动标红）

### 4. 设备管理
- 设备台账管理
- 设备报修（自动将状态改为"维修中"）
- 维修完成（恢复为"正常"）
- 查看维修历史记录

### 5. 人员管理
- 员工信息管理
- 每日考勤签到/签退

### 6. 质量管理
- 质检记录（关联工单、抽检/全检、合格/不合格）
- 质检历史查询

### 7. 系统用户（仅管理员）
- 新增系统用户
- 管理员/普通员工角色设置

## 数据备份
整个系统的数据都在 `db/factory_erp.db` 这一个文件里。
备份方法：直接复制这个文件到安全的地方即可。

## 技术栈
- 后端：Python + Flask
- 前端：Vue 3 + Element Plus（CDN加载，无需安装Node.js）
- 数据库：SQLite（零安装、零配置）

## 注意事项
- 建议定期备份 `db/factory_erp.db` 数据库文件
- 正式使用时请修改默认管理员密码
- 系统需保持运行状态才能访问（关掉命令行窗口即停止）
